    conda install jupyter notebook

    ipython

    2.生成配置文件

     jupyter notebook --generate-config
Writing default config to: /home/daiwenzhu/.jupyter/jupyter_notebook_config.py

    3.修改配置文件内容

    c.NotebookApp.ip = '*'
c.NotebookApp.allow_root = True
c.NotebookApp.open_browser = False
c.NotebookApp.port = 8888
c.NotebookApp.password =
 'sha1:004e6c73a2b8:ff3d9bbc53。。。。。。。。' #输入上面加密后得到的密文
c.ContentsManager.root_dir = '/December/python/jupyter_notebook'
