    • 文件夹推送
      cd *进入要上传的文件夹
      git init #初始化当前文件夹
    • 将本地的文件夹推送到Gitlab
    git remote add origin git@gitdb.*/*git  #关联一个远程库
git push -u origin master     # 推送本地 master 去 origin
git push -u origin dev        # 推送本地 dev  去 origin
    • 如果在本地再进行修改, 然后推送到Gitlab
    git commit -am "change"
git push -u origin master
